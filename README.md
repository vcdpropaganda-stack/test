# VLservice

Base inicial de arquitetura para um marketplace de servicos locais usando Next.js, Tailwind CSS, TypeScript e Supabase.

## Arquivos principais
- `design.md`: direcao visual e tokens basicos do design system.
- `backlog.md`: roadmap inicial por fases.
- `project-structure.md`: estrutura sugerida para o projeto Next.js.
- `prompt.md`: prompt consolidado para geracao assistida de codigo.
- `supabase/schema.sql`: schema inicial com RLS para `admin`, `provider` e `client`.

## Proximo passo recomendado
Bootstrap do app com `create-next-app`, instalar Tailwind/Supabase e aplicar esse schema no projeto.
